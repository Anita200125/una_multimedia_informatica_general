Este proyecto fue realizado como **Trabajo Práctico Final** para la materia **Informática General** de la Universidad Nacional de las Artes (UNA).

## Descripción
El sitio web presenta una propuesta multimedia que combina imagen, video, programación creativa y sonido.  
Incluye materiales originales, collages digitales, y enlaces a videos alojados en YouTube, junto con tres obras interactivas realizadas en **p5.js** por Pablo Ferrando.

Dentro del texto biográfico se incluye además un enlace al sitio **[Surgentes](https://pablo-f61.github.io/SURGENTES/index.html)**, un proyecto colectivo donde Pablo Ferrando forma parte. Se profundiza la relación entre imagen, sonido y espacio digital.

## Contenido
- **index.html**: Página principal del proyecto.  
- **img/**: Carpeta con las imágenes utilizadas.  
- **estilos.css**: Archivo de estilos del sitio.  
- **p5.js/**: Carpetas con los tres proyectos interactivos desarrollados en p5.js.  
- Enlaces a videos en YouTube integrados mediante etiquetas `<iframe>`.  

## Tecnologías utilizadas
- HTML5  
- CSS3  
- JavaScript  
- p5.js  
- Git & GitHub para control de versiones  

## Concepto
La obra propone una exploración entre lo digital y lo sensorial, vinculando la materialidad de la imagen con la programación como lenguaje expresivo.  
Los fragmentos audiovisuales, los collages y las piezas interactivas dialogan en torno a la construcción de la percepción y la memoria.
